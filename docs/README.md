# OctoAcme Project Management Docs

This README serves as a centralized entry point for all core project management processes used by OctoAcme. It includes a summary of practices, roles, and provides direct links to each detailed process document in the `docs/` folder.

## Summary of OctoAcme Project Management Processes

- **Principles:** Customer-first value, iterative delivery, clear ownership, data-informed decisions, psychological safety
- **Core Roles:** Project Manager, Product Manager, Developers, QA/Testing, Stakeholders
- **Lifecycle:** Initiation → Planning → Execution → Release → Retrospective
- **Workflows:** Lean documentation, use of project boards, clear acceptance criteria, risk and dependency management
- **Communication:** Consistent cadence for team and stakeholders, transparent updates, escalation paths
- **Continuous Improvement:** Regular retrospectives, tracking actionable items, and feedback
